# Banking Static Notes – Folder Overview

Ye folder aapke **Banking Static GK / Theory** notes ke liye hai.  
Daily current affairs ke alawa yahan par aap **banks, regulators, schemes, concepts** ka pura theory / history ek jagah maintain kar sakte ho.

---

## 1. Folder ka Purpose

- RBI, NABARD, SBI, RRBs, Cooperative Banks, Payment Banks, Small Finance Banks, etc. – sab ke liye **alag‑alag `.md` files**.
- Har file me:
  - basic intro + definition  
  - history / evolution (important years)  
  - important Acts / amendments  
  - structure / functions  
  - important numbers (capital, shareholding, HQ, taglines – jaha relevant ho)  
  - exam oriented one‑liners + tables (aap baad me khud add/modify kar sakte ho).

Is folder ka main goal ye hai ki **Bank + Regulator + Bank Types ka pure theory** ek jagah ho jise aap baar‑baar revise kar sako.

---

## 2. File Naming Convention

Suggested pattern (aap chahe to follow karo, chahe to apni style use karo):

- Regulator / Institution:
  - `RBI.md`
  - `NABARD.md`
  - `SIDBI.md`
  - `EXIM_Bank.md`
  - `NHB_National_Housing_Bank.md`
- Bank Types:
  - `Types_of_Banks_in_India.md`
  - `Public_Sector_Banks_Overview.md`
  - `Private_Sector_Banks_Overview.md`
  - `Regional_Rural_Banks_RRBs.md`
  - `Cooperative_Banking_Structure.md`
  - `Payments_Banks.md`
  - `Small_Finance_Banks.md`
- Individual Banks (agar aap detail me jaana chahte ho):
  - `SBI_State_Bank_of_India.md`
  - `PNB_Punjab_National_Bank.md`
  - `BOB_Bank_of_Baroda.md`
  - etc.

> Tip: File ka naam aisa rakho ki search karne me easy ho aur turant samajh aa jaye ki file kis ke baare me hai.
